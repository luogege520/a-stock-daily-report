# 报告存储目录

此目录用于存储每日生成的 A股复盘报告。

## 📁 文件命名规则

```
A股晚间复盘报告_YYYY-MM-DD.md
```

例如：
- `A股晚间复盘报告_2026-01-23.md`
- `A股晚间复盘报告_2026-01-24.md`

## 📊 查看报告

### 方法一：GitHub Actions Artifacts

1. 进入仓库的 `Actions` 页面
2. 选择某次运行
3. 在页面底部的 `Artifacts` 部分下载报告

### 方法二：本地运行

如果在本地运行程序，报告会自动保存到此目录。

## 🗂️ 报告管理

### 自动清理

GitHub Actions 的 Artifacts 会在 30 天后自动删除。

如果需要长期保存，建议：
1. 定期下载到本地
2. 上传到云存储
3. 提交到 Git 仓库（不推荐，会增大仓库体积）

### 手动清理

如果报告文件过多，可以手动删除旧文件：

```bash
# 删除 30 天前的报告
find reports/ -name "*.md" -mtime +30 -delete
```
