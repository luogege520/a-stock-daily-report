## 🚀 部署步骤（简化版）

### 第1步：初始化 Git
```bash
cd D:\阶跃AI\a-stock-daily-report
init-git.bat  # Windows
```

### 第2步：创建 GitHub 仓库
访问：https://github.com/new

### 第3步：推送代码
```bash
git remote add origin https://github.com/YOUR_USERNAME/a-stock-daily-report.git
git branch -M main
git push -u origin main
```

### 第4步：配置 Secrets
添加 6 个 Secrets：
- STEPFUN_API_KEY
- SENDER_EMAIL
- SENDER_PASSWORD
- RECIPIENT_EMAIL
- SMTP_SERVER
- SMTP_PORT

### 第5步：启用 Actions 并测试
手动运行一次，检查邮箱
